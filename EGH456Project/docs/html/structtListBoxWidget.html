Detailed Description

The structure that describes a listbox widget.

Member Data Documentation

◆ i16Selected

int16_t tListBoxWidget::i16Selected

The index of the string currently selected in the list box. If no selection has been made, this will be set to 0xFFFF (-1).

◆ i32PointerY

int32_t tListBoxWidget::i32PointerY

The Y coordinate of the last pointer position we received. This is an internal variable used to manage scrolling of the listbox contents and must not be modified by an application using this widget class.

◆ pfnOnChange

void(* tListBoxWidget::pfnOnChange) (tWidget *psWidget, int16_t ui16SelIndex)

A pointer to the application-supplied callback function. This function will be called each time the selected element in the list box changes. The ui16SelIndex parameter contains the index of the selected string in ppcText array or, if no element is selected, 0xFFFF (-1).

◆ ui16OldestEntry

uint16_t tListBoxWidget::ui16OldestEntry

The index of the oldest entry in the ppcText array. This is used by the widget class to determine where to add a new string if the array is full and the listbox has style LISTBOX_STYLE_WRAP. This is an internal variable and must not be modified by an application using this widget class.

◆ ui16Populated

The number of elements in the array pointed to by pci8Text which are currently populated with strings.

◆ ui16Scrolled

uint16_t tListBoxWidget::ui16Scrolled

A flag which we use to determine whether to change the selected element when the pointer is lifted. The listbox will change the selection if no scrolling was performed since the last WIDGET_MSG_PTR_DOWN was received. This is an internal variable and must not be modified by an application using this widget class.

◆ ui16StartEntry

uint16_t tListBoxWidget::ui16StartEntry

The index of the string that appears at the top of the list box. This is used by the widget class to control scrolling of the box content. This is an internal variable and must not be modified by an application using this widget class.
