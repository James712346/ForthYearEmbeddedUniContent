uint16_t tPushButtonWidget::ui16AutoRepeatDelay

The number of pointer events between button presses generated by the auto-repeat function, if PB_STYLE_AUTO_REPEAT is selected. The amount of time to which this corresponds is dependent up on the rate at which pointer events are generated by the pointer driver.
